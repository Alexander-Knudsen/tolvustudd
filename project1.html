Vefsíðugerð og útgáfustýring

Description

This project involved creating a personal portfolio website to showcase my work and achievements. Leveraging my background in website development as a hobby, I utilized HTML, CSS, JavaScript, and GitHub to build and manage the site. I started with an existing site template that I had previously developed with assistance from ChatGPT, which provided a solid foundation for customization tailored to the course requirements.

Technologies Used

  • HTML
  • CSS
  • JavaScript
  • GitHub

Implementation Details

Building on my prior experience, I faced no major challenges during the development of this website. My familiarity with GitHub streamlined the version control and deployment processes, ensuring a smooth workflow. The decision to use a previously created template allowed me to focus on refining and enhancing the website's features to meet the specific demands of the Tölvustudd framleiðsla course.

Features

  • Responsive Design: Ensures optimal viewing experience across various devices and screen sizes.
  • Interactive Navigation: Features a dynamic navigation bar with a dropdown submenu for easy access to individual project pages.
  • Modern Aesthetics: Utilizes a chilled color scheme with deep blue and off-white tones, complemented by sleek typography and subtle animations.
  • Project Showcase: Dedicated sections for featured projects, providing detailed insights into each project's objectives, technologies used, and outcomes.
  • CV Integration: Includes a prominent "View CV" button that links to my comprehensive resume in PDF format.

Conclusion

In conclusion, the "Vefsíðugerð og útgáfustýring" project not only served as a practical application of my web development skills but also provided a platform to effectively present my professional journey and academic accomplishments. This portfolio website stands as a testament to my commitment to continuous learning and my passion for entrepreneurship, particularly through my AI service startup, Botanic ehf.
